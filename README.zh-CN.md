# Eco‑Cat‑Food（公开版）

> 语言：中文 | English: [`README.md`](README.md)

本仓库用于对外公开展示 **Eco‑Cat‑Food** 研发计划的高层视图：以证据 Gate 推进的、面向猫的友善生育控制研究。

## 当前状态
此前的 FSHR‑TMD 小分子路线已在 **Gate 0.5** 判定为 **No‑Start**，现仅作为历史/已取代材料保留，用于审计与复盘。

当前公开状态：
- 当前工作是一个 post‑FSHR 新假说，公开层面仅描述为雄性侧、非内分泌主轴、与精子功能相关。
- 状态：**CONDITIONAL CONTINUE / Pre‑Gate 0.5**。
- 下一步：低披露可行性工作与 Gate 0.5 验证设计。
- 在 IP 与保密策略明确前，公开材料不披露靶点级和执行级细节。

建议从这里开始阅读：[当前公开状态](docs/STATUS_UPDATE_2026-05-30.zh-CN.md)。

## 本仓库是什么 / 不是什么
- 本仓库是：研发治理（规则/流程）与阶段性里程碑进度快照（当前仍处于理论阶段）。
- 本仓库不是：实验数据仓库；不包含化学结构（如 SMILES）、合成路线、配方与剂量策略，也不提供论文全文。

## 安全与伦理
- **不提供任何兽医建议。** 请勿将本仓库内容用于动物治疗/用药指导。
- “猫靶向 / 行为不变 / 特殊人群安全”等表述在公开版中一律按**假设/目标**处理，除非有明确的公开证据支撑，并同时给出不可外推边界。
