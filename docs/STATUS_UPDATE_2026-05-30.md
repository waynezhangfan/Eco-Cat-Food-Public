# Status Update — 2026-05-30

> Language: English | 中文：[`STATUS_UPDATE_2026-05-30.zh-CN.md`](STATUS_UPDATE_2026-05-30.zh-CN.md)

## Summary

The historical FSHR-TMD small-molecule route has been retired as a **Gate 0.5 No-Start**.

Materials in this public repository that discuss the FSHR small-molecule route are retained for transparency and auditability. They no longer represent the active development path.

The project is now evaluating a new post-FSHR hypothesis at **CONDITIONAL CONTINUE / Pre-Gate 0.5**. At the public level, this hypothesis is described only as:

- male-side
- non-endocrine
- sperm-function-related
- still awaiting functional validation

## What Is Not Claimed

At this stage, the project does not claim:

- validated efficacy
- cat-specificity
- dog or non-target safety
- oral feasibility
- field readiness
- product readiness
- investment readiness

## Disclosure Boundary

Target-level and execution-level details are withheld pending IP and confidentiality decisions.

Public materials will not disclose:

- target names or target families
- sites, residues, pores, pockets, motifs, or binding-region details
- tool compounds, scaffolds, SAR, or concentration ranges
- assay protocols or route-specific pass/fail thresholds
- PK/ADME route hypotheses
- CRO, sample-logistics, collaborator, or unpublished dataset details

## Use of Historical Materials

The non-confidential v0.9 technical package dated 2026-02-10 is now a **superseded historical package**. It may be useful for understanding the prior governance process and evidence discipline, but it should not be shared as the current technical package for the project.
