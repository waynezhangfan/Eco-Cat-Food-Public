# Evidence Policy (Public)

> Language: English | 中文：[`EVIDENCE_POLICY.zh-CN.md`](EVIDENCE_POLICY.zh-CN.md)

## Evidence levels
- **Strong** — directly relevant, reproducible, and bounded (minimal extrapolation)
- **Medium** — source is verifiable but has meaningful mechanism/species/context gaps; supports directionality or methodological constraints only
- **Weak** — indirect or review-level background; not used for key go/no-go decisions

## Citation rules
- Public repo includes metadata (DOI/title) and project-authored policies/notes only.
- Do not cite unstable line anchors; avoid implying internal-only data exists.
- For every key claim: state evidence level + non‑extrapolation boundaries.
