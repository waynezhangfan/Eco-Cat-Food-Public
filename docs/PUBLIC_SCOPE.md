# Public Scope

> Language: English | 中文：[`PUBLIC_SCOPE.zh-CN.md`](PUBLIC_SCOPE.zh-CN.md)

## Included (public)
- High-level stage definitions and milestone progress
- Evidence grading and citation policy
- Literature metadata list (DOI/title/year/venue)

## Excluded (never public, or only under NDA)
- Chemical structures/SMILES, SAR tables, substitution maps, synthesis routes
- Formulations, concentrations, dosing strategies, manufacturing parameters
- Raw experimental data, lab notebooks, CRO SOWs/quotes, vendor details
- Full‑text papers (PDF/HTML), screenshots, extracted figures
- For the current post‑FSHR hypothesis: target names or target families, sites/residues, pores/pockets, motifs, binding-region details, tool compounds, assay protocols, route-specific pass/fail thresholds, PK/ADME route hypotheses, sample logistics, collaborator details, or unpublished datasets

## External-claim red lines (public wording)
Statements like “no behavioral impact”, “safe for kittens/pregnancy/lactation”, or “no impact on other species” must be treated as hypotheses unless supported by explicit public evidence with bounded non‑extrapolation notes.

For the current post‑FSHR hypothesis, the public repository only states the mechanism class and Gate status. It does not claim validated efficacy, cat-specificity, dog or non-target safety, oral feasibility, field readiness, product readiness, or investment readiness.
