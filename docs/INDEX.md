# Index

This public repository is a curated, read‑only view focused on:
- Stage definitions and milestone progress
- Evidence grading & citation policy
- A lightweight reference list (DOI/title metadata)

> Language: English | 中文：[`INDEX.zh-CN.md`](INDEX.zh-CN.md)

## Documents
- `PROGRESS.md` — stage definitions and current status
- `ROADMAP.md` — next milestones (high level)
- `PUBLIC_SCOPE.md` — what is included/excluded in the public version
- `EVIDENCE_POLICY.md` — evidence levels and citation rules
- `REFERENCES.tsv` / `REFERENCES.csv` — literature metadata (no full texts)
- `UPDATING.md` — how this public repo is updated
- `SHARING.md` — what is safe to share externally

## Investor / partner packages
- `nonconfidential/00-PACKAGE_INDEX.NC.md` — shareable non‑confidential technical package (English default + Chinese link)

> Note: `docs/_confidential_DO_NOT_SHARE/` contains NDA-only drafts for later use. It is intentionally excluded from public commits via `.gitignore`.
