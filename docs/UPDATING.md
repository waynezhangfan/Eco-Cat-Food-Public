# Updating This Public Repository

> Language: English | 中文：[`UPDATING.zh-CN.md`](UPDATING.zh-CN.md)

This repo is updated **only at milestone freezes** or explicit public status corrections.

Recommended workflow:
1) Draft the public update content (README / docs/* / reference list)
2) Run a quick self-check:
   - no PDFs/images are added
   - no local paths or internal-only references are mentioned
   - `docs/_confidential_DO_NOT_SHARE/` is NOT included in commits
   - superseded packages are clearly labeled as historical
   - target-level and execution-level details remain private
3) Commit and push

Public repo intentionally stays minimal; sensitive technical details remain private.
