# 公开范围

> 语言：中文 | English: [`PUBLIC_SCOPE.md`](PUBLIC_SCOPE.md)

## 公开内容（Included）
- 高层阶段定义与里程碑进度
- 证据等级与引用口径
- 文献元数据列表（DOI/标题/年份/期刊）

## 永不公开（或仅在 NDA 下提供）
- 化学结构/SMILES、SAR 表、取代基地图、合成路线
- 配方、含量/浓度、剂量策略、生产/工艺参数
- 原始实验数据、实验记录本、CRO SOW/报价、供应商信息
- 论文全文（PDF/HTML）、截图、提取的图表
- 对当前 post‑FSHR 新假说：靶点名或靶点家族、位点/残基、孔区/口袋、基序、结合区域细节、工具化合物、实验方案、与具体路线绑定的 pass/fail 阈值、PK/ADME 路线假说、样本物流、合作方细节或未公开数据集

## 对外主张红线（公开表述）
诸如“行为不变”“幼猫/孕期/哺乳期安全”“对其他物种无影响”等表述，在公开版中必须作为**假设/目标**处理，除非有明确的公开证据支撑，并同时给出不可外推边界。

对于当前 post‑FSHR 新假说，公开仓库只陈述机制类别和 Gate 状态；不主张已验证有效性、猫特异性、犬或非目标物种安全性、口服可行性、外场可用性、产品就绪或投资就绪。
